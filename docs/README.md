# Documentation

## Structure

```
docs/
├── README.md              ← this file
├── ARCHITECTURE.md        ← system architecture overview
└── decisions/             ← Architecture Decision Records (ADRs)
    ├── 01-ruah-analysis.md
    ├── 02-mempalace-analysis.md
    └── 03-turboquant-analysis.md
```

## Conventions

- **decisions/** — technical decisions with context, alternatives, and rationale
- Markdown files, kebab-case names with numeric prefix
- Content in English, code and commands in English
